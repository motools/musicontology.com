Getting started

The Music Ontology provides a model for publishing structured music-related data on your web site or through your API. If you're unsure about why you should publish music-related data in this way, please read our background notes. Below are a few typical examples of using the Music Ontology, in three different syntaxes: RDFa in HTML for embedding music-related data on a web page, and JSON-LD and Turtle for exposing music-related data as part of an API.

Adding prefixes

In order to start using the Music Ontology, you will need to add a few prefixes to your document. This is done by adding the following to your document.

In HTML:

<html prefix="mo: http://purl.org/ontology/mo/
              dc: http://purl.org/dc/elements/1.1/
              foaf: http://xmlns.com/foaf/0.1/">
  ...

<html>

In JSON-LD:

{
  "@context": {
    "mo": "http://purl.org/ontology/mo/",
    "dc": "http://purl.org/dc/elements/1.1/",
    "foaf": "http://xmlns.com/foaf/0.1/"
  }
  ...
}

In RDF/Turtle:

@prefix mo: <http://purl.org/ontology/mo/> .
@prefix dc: <http://purl.org/dc/elements/1.1/> .
@prefix foaf: <http://xmlns.com/foaf/0.1/> .
...

Describing a track

The following example shows how to describe a track, the band who made it, and linking the band to its MusicBrainz identifier.

In HTML:

<span typeof="mo:Track" about="#track-1">
  <span property="dc:title">Turnover</span>
  is a track by 
  <a rel="foaf:maker" typeof="mo:MusicGroup" 
     href="http://musicbrainz.org/artist/233fc3f3-6de2-465c-985e-e721dbabbace#_">
    <span property="foaf:name">Fugazi</span>
  </a>
</span>

In JSON-LD:

{
  "@id": "http://example.org/#track1",
  "@type": "mo:Track",
  "dc:title": "Turnover",
  "foaf:maker": {
    "@id": "http://musicbrainz.org/artist/233fc3f3-6de2-465c-985e-e721dbabbace#_",
    "@type": "mo:MusicGroup",
    "foaf:name": "Fugazi"
  }
}

In RDF/Turtle

<#track-1>
  a mo:Track ;
  dc:title "Turnover" ;
  foaf:maker <http://musicbrainz.org/artist/233fc3f3-6de2-465c-985e-e721dbabbace#_> .
<http://musicbrainz.org/artist/233fc3f3-6de2-465c-985e-e721dbabbace>
  a mo:MusicGroup ;
  foaf:name "Fugazi" .

Describing a band

The following example shows how to describe a band and its members.

In HTML:

        
<span typeof="mo:MusicGroup" about="#band-1">
  <span property="dc:title">Fugazi</span>
  has the following members:
  <ul rel="foaf:member">  
    <li about="#imk" typeof="mo:MusicArtist"><span property="foaf:name">Ian MacKaye</span></li>
    <li about="#jl" typeof="mo:MusicArtist"><span property="foaf:name">Joe Lally</span></li>
    <li about="#gp" typeof="mo:MusicArtist"><span property="foaf:name">Guy Picciotto</span></li>
    <li about="#bc" typeof="mo:MusicArtist"><span property="foaf:name">Brendan Canty</span></li>
  </ul>
</span>

In JSON-LD:

{
  "@id": "http://example.org/#band-1",
  "@type": "mo:MusicGroup",
  "dc:title": "Fugazi",
  "foaf:member": [{
    "@id": "http://example.org/#imk",
    "@type": "mo:MusicArtist",
    "foaf:name": "Fugazi"
  },{
    "@id": "http://example.org/#jl",
    "@type": "mo:MusicArtist",
    "foaf:name": "Joe Lally"
  },{
    "@id": "http://example.org/#gp",
    "@type": "mo:MusicArtist",
    "foaf:name": "Guy Picciotto"
  },{
    "@id": "http://example.org/#bc",
    "@type": "mo:MusicArtist",
    "foaf:name": "Brendan Canty"
  }]
}

In RDF/Turtle:

<#band-1>
   rdf:type mo:MusicGroup;
   dc:title "Fugazi";
   foaf:member <#imk>;
   foaf:member <#jl>;
   foaf:member <#gp>;
   foaf:member <#bc> .
<#imk>
   rdf:type mo:MusicArtist;
   dc:title "Ian MacKaye" .
<#jl>
   rdf:type mo:MusicArtist;
   dc:title "Joe Lally" .
<#gp>
   rdf:type mo:MusicArtist;
   dc:title "Guy Picciotto" .
<#bc>
   rdf:type mo:MusicArtist;
   dc:title "Brendan Canty" .

The following example shows how to associate the band members with a period of time and an instrument.

In HTML:

<span prefix="ical: http://www.w3.org/2002/12/cal/ical#" typeof="mo:MusicGroup" about="#band-1">
  <span property="dc:title">Fugazi</span>
  has the following members:
  <ul rel="mo:membership">
    <li typeof="mo:Membership">
      <span rel="mo:artist"><span about="#bc" typeof="mo:MusicArtist" property="foaf:name">Brendan Canty</span></span> played
      <a rel="mo:primary_instrument" typeof="mo:Instrument" href="http://dbpedia.org/resource/Drum_kit">drums</a>
      from <span property="ical:dtstart" value="1987-01-01T00:00:00Z">1987</span>
    </li>
  </ul>
</span>

In JSON-LD:

{
  "@id": "http://example.org/#band-1",
  "@type": "mo:MusicGroup",
  "dc:title": "Fugazi",
  "mo:membership": [{
    "@type": "mo:MusicMembership",
    "mo:artist": {
      "@type": "mo:MusicArtist",
      "foaf:name": "Brendan Canty"
    },
    "mo:primary_instrument": {
      "@type": "mo:Instrument",
      "@id": "http://dbpedia.org/resource/Drum_kit"
    }
  }]
}

In RDF/Turtle:

<#band-1>
   rdf:type mo:MusicGroup;
   dc:title "Fugazi";
   mo:membership [
     rdf:type mo:Membership;
     mo:artist <#bc>;
    mo:primary_instrument <http://dbpedia.org/resource/Drum_kit>;
   <http://www.w3.org/2002/12/cal/ical#dtstart> "1987-01-01T00:00:00Z" .
<#bc>
   rdf:type mo:MusicArtist;
   foaf:name "Brendan Canty" .
<http://dbpedia.org/resource/Drum_kit>
   rdf:type mo:Instrument .
